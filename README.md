# page-loader
utility for downloading web pages
## installing
```
pip install --index-url https://test.pypi.org/simple --extra-index-url https://pypi.org/simple alekorn-page-loader
```
## usage
```
page-loader [URL] -o [PATH] -l [LOG LEVEL]

LOG LEVEL: 
debug
info
warning
eror
critical
```
